    After reviewing the complaints about my initial proposal, I did some thinking and some research. With all this in mind, I have decided to amend my proposal.

    RFC #Nine-Two-Nine b: Classify Certain Type-Changing Moves as Chainable, Presented by Keys2tkingdom

    Since they act upon the user rather than imparting a condition on their opponent or the battlefield, similar to other self-buff moves, I request that the Status moves Conversion, Conversion 2, Camouflage, and Reflect Type be treated as status buff moves for chaining purposes. The "effect dissipates after the chained-into move is used" rule would be in full effect.

    The wiki battle mechanics page would be updated to note this classification.

    To address Newf's concerns of RNG abuse on the Conversion moves, I've reworded descriptions to show RNG mechanics in moves they exist in and to show how moves are restricted in moves that lack them.

    Unfortunately, I can not directly address Bitter's and Zeal's complaints beyond the 'giving a watered-down Protean to Porygon-line mon' complaint. And I'm addressing that complaint by making all type-changing moves that only affect the user included in this proposal.

    Chaining the moves in a responsible fashion and not abusing them will help with Verax's concerns as well as helping with Newf's concern as well.

    The reasoning I have for this request is the following:

    • Conversion: Conversion induces a type change in the user to allow for a possible STAB - as it relies on the typing of the first move the mon used in battle (In a similar way why one would have reasons to chain Swords Dance or Calm Mind)

    • Conversion 2: Conversion 2 induces a random type change in the user to allow it to be resistant to the typing of last move used by a mon the user is facing. (In a similar way one would have reasons for chaining a Protect move)

    • Camouflage: Camouflage induces a specific type change depending in the environment the user is in. (As the type changes are set this move could be used for similar reasons to chain moves like Swords Dance and/or Protect)

    • Reflect Type: Reflect Type induces a type change in the user that makes the user's typing the same as the mon it's facing - e.g. the user becomes Fire/Dragon/Grass if facing a Mega Charizard X affected by Forest's Curse (As types tend to resist or be neutral to themselves, this move would more likely be used for reasons one would have for chaining a Protect move)

    Since Bitter and Zeal - and possibly a few others - are worried about giving a pseudo-Protean to a small pool of mon, I'm going to make a few extra points here:
