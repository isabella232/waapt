    Verax: I wouldn't be entirely opposed, and it might turn out well, though I think creating an entirely different region might be a bit much. It takes a lot of time to develop these things from the ground up and spending two or three months or so in a region like that might cause us to run into some problems.

    Umbra: ...You want me to use a fan region for the TTYD Arc?

    Well, that might fix some problems, but it could create some others.

    If I didn't use a fanmade region, it would probably cause a few changes to locations in Hoenn, Sinnoh, Unova, and maybe other regions, though I'd try my best to make such changes as inconsequential as I could. The only places that I think would get much of a change would be Rogueport and Keelhaul Key's equivalents. Plus a floating fighting arena. (Oh, and the Moon, but we're probably never going there again anyway, and that one's kinda inevitable.)

    If I went fanmade, I wouldn't need to stretch certain story details as much, but it opens up some other things that I think might be problems...

    For one, I think this kind of works best if the NP Cs are Pokemon living in human settlements, which I fear I may not portray quite right if I have to build a region from the ground up. (Namely that humans exist here, but aren't really important to the plot, so it might seem like they're not *really* there. I feel a bit less iffy about running into this problem with a canon region, but that might just be me.)

    There's the option of having more important humans, but then I'd need to think up interesting Pokemon for them, and I'm not entirely sure how well I'd do with that. There's also the option of making this region Pokemon-dominated, but I don't think that'll work in the OU, and running it in other Pokemon dominant verses might run us into some problems, especially because with how this is set up, I want to use the Alakagross specifically for it.

    Plus there's the aversion of others to noncanon regions and I'm admittedly a bit averse to making a whole region just for this plot unless a bunch of people think it would be better than in the canon regions.

    Admittedly, none of these might be as big of problems as I think they are, but if I did it in a noncanon region, my guess is I'd have to do a fair amount more planning and tweaking. I'm not entirely opposed to it if people would prefer I try something like that, though.

    Anyone have thoughts on this?

    Psyche: Is Ransei!Jace a Xatu? If so I don't see a problem, Xatu have really long lives.

    Verax: While the idea sounds good, my immediate reaction to a fanmade region is that yeah, it doesn't work.

    However, this is not all-encompassing.

    First off, I have nothing against the existence of fanmade regions, as clearly the Pokeworld exists outside of the places given us by the games, anime, TCG, and manga. I think they should be kept to a minimum to avoid making large patches where we could theoretically overstep into possible future canon.

    Second, I would say that any sort of region that isn't an existing one would need to have some sort of draw, a reason why we can't use a preexisting one. Not just new gyms or new cities, but something that makes it so it isn't the kind of place you'd set a main series game in. For instance, Holon is different because of Deltas, the magnetic storms, and the prevalence of scientists.

    As such, the idea of a fanmade region, unless it's being used for some purpose other than "we go to a new region, battle the Gyms, catch mons, fight the bad guys", is not one that really interests me. Sorry.
