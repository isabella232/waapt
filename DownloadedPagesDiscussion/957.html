    The way I see [Choreography] working is synchronized movement and/or Dance Battling.

    Obviously, the basic mechanics of [Choreography] work identically to Round - the base power (BP) doubles if a mon uses it in conjunction with another.

    As an Optional mechanic, [Choreography] can be used as a part of a "Round combo" without a base power penalty.

    Examples:
    • Round #1 is 60 BP => [Choreography] #1 is 120 BP => [Choreography] #2 is 240 BP

    • Round #1 is 60 BP => [Choreography] #1 is 120 BP => Round #2 is 240 BP

    • etc.
